总则

尽管我们可能会收集信息以改善应用程序和服务质量，但是我们非常重视您的隐私。为了更好的保护您的隐私，我们提供了本隐私政策声明，描述我们收集、处理数据的方式，使用的部分技术等内容。

如何阅读本隐私政策

隐私政策通常冗长且难以阅读。我们会对其中一些内容进行标注，便于您的阅读。

标注为 绿色 的内容对您有利。

标注为 黄色 的内容可能需要您额外注意。

标注为 红色 的内容需要您额外注意，因为有可能对您不利。

标记为 加粗 的内容需要您额外注意，因为有可能被您忽视。

修订策略

在法律允许的范围内，我们保留在任何时间修改，修订。解释或更新本政策的权利，可能不会通知到您。

您应该定期访问此网页，使用浏览器上的 刷新 按钮，检查本政策的任何修改。如果在点击浏览器的 刷新 按钮后， 更新记录 保持不变，则可以假设，没有更改。 更新记录 的变化表示该政策已被更新或编辑，更新或编辑的版本在发布后立即取代先前任何版本。浏览器、运营商、CDN、和其他组织或个人可能出于各种因素缓存我们的页面，在这种情况下导致您看到的页面内容和实际不符，应当以我们提供的为准。

我们是谁

我们的站点地址是：https://www.jiecs.top，我们提供的服务包括但不限于该博客网站，在线服务，交互式应用程序。

我们收集何种数据

我们会收集您的数据用于服务所必须或改善服务体验。

统计服务

我们会主动使用统计服务通过我们提供的服务来收集基本不包含指向性的匿名化数据。

统计服务：同时使用 Microsoft Clarity、Google Analytics、百度统计 。杰出兽安卓APP 中使用 百度移动统计

数据包括但不限于：

  • 使用我们提供的服务时的 IP 地址以及 IP 地址可能提供的网络运营商和地理位置等

  • User Agent 可能提供的浏览设备、操作系统、浏览器，包含具体版本号，但不包含完整 User Agent

  • 浏览我们提供的服务时的具体操作回放（仅本博客）

  • 应用程序和网站的启动、加载速度，关键的运行计时

  • 引荐来源、访问频率、精准到秒的访问日期和时间、行为流

这些数据可能可以互相关联，这意味着我们能够获取类似以下内容：

一位广东电信新访客于 19:30:10 使用 Pixel 3，Chrome 浏览器通过谷歌搜索进入一篇文章，加载耗时 500ms，向下滚动浏览数秒后通过导航栏返回首页，若干秒后关闭页面

Do Not Track

本博客尊重 Do Not Track 请求

如果您不希望上述跟踪器应用于您，可以在浏览器设置中启用 Do Not Track（或被称为“不跟踪”），并可在页脚处见到相应提示。

交互行为

如果您在使用我们的服务时发生交互行为（如评论），我们可能会向您收集以下数据：

  • 您主动提供的昵称、电子邮箱地址、与您相关的网址

  • 根据您所提供的电子邮箱地址获取到的公开的部分社交媒体头像（无法定位至具体账户）

  • User Agent 可能提供的浏览设备、操作系统、浏览器，包含具体版本号，但不包含完整 User Agent

  • 公开发表的内容（包括在本博客网站发布的评论等）

我们从哪些第三方获取何种数据

我们会主动通过第三方获取完全不包含指向性的匿名化数据用于改善服务体验。

我们获取的这些信息都由第三方严格处理，仅有大致范围和比例，完全不包含针对个体的指向性

统计服务所附加的数据

部分统计服务会基于其收集的数据提供比上述更多的数据

这些数据包括但不限于：

  • 受众的性别、年龄范围比例
  • 受众所感兴趣的大分类比例（如科技、游戏、媒体）

我们如何处理数据

我们会处理您的数据用于服务所必须或改善服务体验。

共享

公开

您在使用我们的服务时发生交互行为（如评论）时我们收集的数据部分数据将公开在博客网站上，包括：

  • 您主动填写的昵称、与您相关的网址

  • 您主动提供的电子邮箱地址所生成所生成的匿名化字符串（又称为哈希）

  • 公开发表的内容（包括在本博客网站发布的评论等）

第三方

我们会将部分数据共享给第三方，包括：

  • 您主动提供的电子邮箱地址所生成所生成的匿名化字符串（又称为哈希）将提供给 Cravatar 服务以确认您是否有合适的头像

  • 您主动在评论内容中附上的图片将提供给水墨图床作为图片存储

  • 公开发表的评论相关数据将提供给 Akismet 服务以自动确认您所发布的内容是否是垃圾消息

保留

如果您在本博客网站留下评论，相关数据通常会被无限期保存。（但我们不提供存储服务，不保证数据的稳定安全）

统计数据会被保留一年后销毁。

您对您的数据有什么权利

如果您有本博客网站评论内容和相关数据，您可以要求我们抹除所有关于您的个人数据。

这不包括我们因管理、法规或安全需要而必须保留的数据。

我们使用的技术

Cookies

  • 当您在本博客上评论时我们会将您主动填写的昵称、与您相关的网址存储在 Cookie 中记住。

  • 我们使用的一些统计服务会留下 Cookies 用以记录数据。

其他站点的嵌入内容

此站点上的文章可能会包含嵌入的内容（如视频、图像、文章等）。来自其他站点的嵌入内容的行为和您直接访问这些其他站点没有区别。

这些站点可能会收集关于您的数据、使用 Cookies、嵌入额外的第三方跟踪程序及监视您与这些嵌入内容的交互，包括在您有这些站点的账户并登录了这些站点时，跟踪您与嵌入内容的交互。
